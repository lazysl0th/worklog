# Журнал работ на строительном объекте

Тестовое задание для автоматизации учета и контроля строительных работ.

## 🛠 Технологический стек

### Бэкенд (backend/)
- **Платформа:** Node.js, TypeScript, Express
- **База данных:** PostgreSQL + Prisma ORM
- **Архитектура:** Clean Architecture (Domain, Application, Infrastructure layers)
- **Валидация данных:** Zod
- **Аутентификация:** Passport.js (JWT в httpOnly cookies)

### Фронтенд (frontend/)
- **Фреймворк:** React, TypeScript
- **Стейт-менеджмент:** Redux Toolkit
- **Стилизация:** Tailwind CSS
- **Архитектура:** Feature-Sliced Design (FSD)

---

## 🏗 Структура проекта

Проект организован в виде монорепозитория:
- `backend/` — Серверная часть приложения (Clean Architecture)
- `frontend/` — Клиентская часть приложения (FSD)

---

## ⚡ Особенности реализации и качества кода
- **Строгая типизация:** Полный запрет на использование типа `any` и приведения типов через `as` (настроено на уровне ESLint).
- **Валидация:** Строгая проверка входящих данных и конфигурации окружения.
